# Prerequisites for ICAI Tax Alerts App

1. **Git** – https://git-scm.com/downloads  
2. **Node.js (>=16.x)** – https://nodejs.org/  
3. **Firebase CLI** – 
pm install -g firebase-tools  
4. **Java JDK (>=11)** – https://adoptopenjdk.net/  
5. **Android Studio** – https://developer.android.com/studio  
6. **Android SDK & Gradle** (installed via Android Studio)  
7. **Jsoup** (will be added via Gradle)  
8. **Perplexity Pro** account (for prompt-engineering)
